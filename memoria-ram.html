Memória ram

O que faz? A memória RAM (Memória de Acesso Aleatório) armazena temporariamente os dados que o processador precisa acessar rapidamente enquanto o computador está em uso. Diferente do armazenamento permanente, como discos rígidos ou SSDs, a RAM é volátil, o que significa que seus dados são apagados quando o computador é desligado. Seu principal papel é garantir que o sistema e os programas rodem de forma rápida e eficiente. Quando foi criado? A primeira forma de RAM (SRAM) foi inventada em 1968. Quem inventou? Robert Dennard é creditado pela invenção da DRAM.
